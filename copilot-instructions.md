# 指示書
## 要求
- マインクラフトのMODを作成してほしい
- MODの名前は「micrafantasy」にしてほしい
- MODのバージョンは1.0.0にしてほしい
- MODの対応バージョンは1.21.11にしてほしい
- MODのテーマは、FF14のジョブシステムをマインクラフトに実装すること
- ジョブシステムを実装するために、以下の要素を実装してほしい
- プロジェクトのpackage名は「cloud.laboratory.n.micrafantasy」にしてほしい
- クリスタルをイメージしたブロックを追加してほしい
- ブロックからはジョブストーンを１つ獲得できる
- ジョブストーンには複数種類がありブロックを壊すとランダムで獲得できる
- ジョブストーンはジョブごとのジョブストーンの総称である
- ジョブストーンのイメージは、FF14のジョブストーン、またはジョブアイコンを参考にして画像を作成してほしい
  - https://www.etsy.com/jp/listing/1010386549/fainarufantajxivwoimjishitajobusutnsouru
- 今回実装するジョブストーン
  - ナイト
- ジョブは今後増やしていく予定である
- 各ジョブ、スキルは10種類ずつ実装する予定である
- 各ジョブ、特性は10種類ずつ実装する予定である
- ジョブのスキルスロットは画面上部に配置してほしい
- スキルには、物理スキルと魔法スキルとリミットブレイクとジョブスキルがある
- 物理スキルはスタミナを消費するので、スタミナを実装してほしい
- 魔法スキルはマナを消費するので、マナを実装してほしい
- リミットブレイクはリミットゲージを消費するので、リミットゲージを実装してほしい
- ジョブスキルは、１日に１回使用できるようにしてほしい
- 与ダメージを伴うリミットブレイク・魔法スキル・ジョブスキルは、その与ダメージの1/10のダメージを地形に与えるようにしてほしい
- ブロックのダメージを
- スタミナもマナも、時間経過で回復するようにしてほしい
- リミットゲージは、敵を攻撃することで回復するようにしてほしい
- リミットゲージが最大値になると、リミットブレイクを使用できるようにしてほしい
- スタミナは、食事をとることで回復するようにしてほしい
- マナは、睡眠をとることで回復するようにしてほしい
- スタミナは、ジョブのレベルが上がることで最大値が増えるようにしてほしい
- マナは、ジョブのレベルが上がることで最大値が増えるようにしてほしい
- ジョブのレベルは、経験値を獲得することで上がるようにしてほしい
- 経験値は、敵を倒すことで獲得できるようにしてほしい
- ジョブのレベルが上がると、スキルや特性を習得できるようにしてほしい
- ナイトのジョブストーンを装備するとナイトにジョブチェンジすることができる
- ジョブストーン専用の枠を作ってほしい
- ナイトにジョブチェンジすると、ナイトのスキル（アクティブスキル）を使用できるようになる
- ナイトにジョブチェンジすると、ナイトの特性（パッシブスキル）を使用できるようになる
- ジョブの概要はFF14を参考にする
- ジョブのスキルはFF14を参考にする
- ジョブの特性はFF14を参考にする
- スキルスロットに表示する画像は、FF14のスキルアイコンを参考にして画像を作成してほしい
  - https://jp.finalfantasyxiv.com/jobguide/paladin/
- ナイトについて
  - スキル
    - 0: 通常攻撃（物理スキル：単体攻撃）
    - .: 通常防御（物理スキル：単体防御）
    - 1: 挑発（物理スキル：敵対心アップ＋移動速度ダウン付与）
    - 2: シールド・バッシュ（物理スキル：敵対心アップ＋スタン）
    - 3: クレメンシー（魔法スキル：回復＋継続回復）
    - 4: ファスト・ブレード（物理スキル：範囲小攻撃＋攻撃速度アップ）
    - 5: ライオットソード（物理スキル：範囲攻撃＋攻撃力アップ）
    - 6: レイシ・オブ・ハルオーネ（物理スキル：範囲大攻撃）
    - 7: センチネル（魔法スキル：敵対心アップ＋大軽減）
    - 8: インビンシブル（ジョブスキル：敵対心アップ＋無敵）
    - 9: ラスト・バスティオン（リミットブレイク：敵対心アップ＋超攻撃＋超軽減＋超バフ）
  - 特性は後日実装予定
- 操作性
  - スキルは、NUMBERキー（0-9）で使用できるようにしてほしい
  - キーボードでカメラワークを操作できるようにしてほしい
- その他
  - debugモードを実装してほしい
  - debugモードでは、全てのジョブストーンを獲得できるようにしてほしい
    - debugモードでは、開始時・死亡時に全てのジョブストーンを獲得できるようにしてほしい
  - debugモードでは、全てのスキルを使用できるようにしてほしい
  - debugモードでは、全ての特性を使用できるようにしてほしい
  - debugモードは、configファイルでON/OFFできるようにしてほしい
  - configファイルは、MODのフォルダ内に「config.properties」という名前で作成してほしい
